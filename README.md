# InterCiter

A provenance-first knowledge-graph system that extracts claims from scientific papers, anchors them to exact source passages, and classifies how citing claims relate to cited work.

- **Schema:** [schema/interciter.yaml](schema/interciter.yaml) — the logical data model as an executable, BioLink-aligned [LinkML](https://linkml.io) schema
